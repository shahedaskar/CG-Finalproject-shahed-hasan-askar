لعبة الجري اللانهائي - OpenGL

🎮 فكرة اللعبة

هذه لعبة ثلاثية الأبعاد بسيطة باستخدام OpenGL.
يتحكم اللاعب بمكعب يتحرك على طريق، ويجب عليه تجنب العوائق وجمع العملات.

🕹 عناصر التحكم
- السهم لليسار ← → التحرك لليسار
- السهم لليمين → → التحرك لليمين

⚙️ طريقة التشغيل

1. افتح المشروع باستخدام Visual Studio
2. تأكد من تثبيت المكتبات التالية:
   - GLEW
   - GLFW
   - GLM
   - stb_image
3. شغّل المشروع

📁 مكونات المشروع

- Source → يحتوي على كود البرنامج (main.cpp)
- Shaders → ملفات الـ shaders
- Assets → الصور (مثل صورة الطريق)
